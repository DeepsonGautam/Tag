# Tag It

Welcome to **Tag It**.

This is a local 2-player game where the goal is pretty simple: don't be the one who's **tagged** when the timer runs out.

## Controls

**Player 1**

* W A S D

**Player 2**

* Arrow Keys

## How to Play

The match lasts **120 seconds**.

One player starts with the **Tagged** marker above their head. Your job is to make sure the other player has it before time runs out.

When the timer reaches 0:

* The player with the **Tagged** marker loses.
* The other player wins.

That's it. Run around, chase each other, and try not to panic.

## Features

* Local multiplayer for 2 players.
* Dynamic camera movement that keeps up with the action.
* Sound effects that make everything a little more satisfying.


## Final Words

This game is best played against a friend. If you win, you're just better. If you lose... the game was probably rigged anyway.

Have fun!
